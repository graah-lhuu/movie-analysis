# 🎬电影数据分析项目

## 📋项目简介
使用Python和Jupyter Notebook对电影数据集进行探索性数据分析(EDA)，包含数据清洗、可视化、统计分析和洞察发现。

## 🚀快速开始

### 环境要求
bash pip install pandas numpy matplotlib seaborn jupyter

### 文件结构
movie-analysis/  
├── notebooks/movie-analysis.ipynb    # 主分析笔记本  
├── data/raw/movie_metadata.csv      # 原始数据  
├── src/utils.py                     # 工具函数  
└── README.md

### 运行步骤
1.将数据文件放入 data/raw/目录  
2.打开 notebooks/movie-analysis.ipynb  
3.逐个运行单元格或运行全部  

## 📊分析内容
数据清洗与预处理  
IMDb评分分布分析  
导演表现统计  
电影类型趋势  
国家/地区对比  
时长与评分关系  
财务数据分析  
特征相关性分析  

## 🎯主要功能
✅ 自动数据加载与清洗  
✅ 多种可视化图表  
✅ 统计分析报告  
✅ 结果导出功能  
✅ 错误处理与日志

清洗后的数据集
